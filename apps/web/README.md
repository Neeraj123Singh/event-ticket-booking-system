# Web (`apps/web`)

Next.js 15 App Router UI for the ticketing platform.

## Run

From the **monorepo root** (see root [README.md](../../README.md)):

```bash
pnpm dev
```

Dev server listens on **http://localhost:3001** (port 3001 avoids clashes with other tools on 3000).

## Env

Copy [`.env.example`](./.env.example) to `.env.local` if you need to override API URLs for the browser or server.
